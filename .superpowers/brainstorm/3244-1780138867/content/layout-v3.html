Modularity

The principle of partitioning a system into discrete, interchangeable components that can be developed, tested, and maintained independently.

Importance: core

How xyflow Handles This

In @xyflow/react, each custom node component defines Handle components at specific positions (top, right, bottom, left). Edges automatically connect from handle to handle — never through node bodies.

// CustomDocumentNode.tsx — Handles on borders, not center
function CustomDocumentNode({ data }: NodeProps) {
  return (
    <div className={styles.node}>
      <Handle type="source" position={Position.Right} />
      <Handle type="target" position={Position.Left} />
      <span>{data.label}</span>
    </div>
  );
}
